# React-do-Zero

## 1 - Introdução:
  ### O que é o React: 
  - É uma **biblioteca JavaScript** para desenvolvimento de aplicações front-end;
    - biblioteca vs framework:
    - Na documentação o propior react se chama de biblioteca;
    - Quando trabalhamos com um framework, como por exemplo o Laravel, tem uma forma de se construir, se não for seguido corretamente a aplicação pode não funcionar;
    - Agora com a biblioteca se tem mais liberdade, podendo-se por apenas uma pequena parte em um site, e não fazer o site inteiro com o React;
  - São **SPA**(Single Page Application);
    - É uma pagina que renderiza todas as páginas da aplicação numa página só;
    - Tem um carregamento só, e depois apenas requisições assincronas, apenas quando é requisitado algo do servidor;
  - Arquitetura Baseada em Componentes;
    - Importante para a reutilização dos componentes
  - Pode ser inserido na aplicação ou a aplicação toda pode ser em React;
    - Da pra usar um cdn e utilizar o react;
  - Mantido pela Meta;
